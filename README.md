# Sorting Visualisation

A React visualisation of the following algorithms:
* Merge Sort
* Quick Sort
* Bubble Sort
* Selection Sort
* Insertion Sort

It allows you to create different sizes of array and adjust the animation speed and enjoy the show!
 
# Requirements

* React.js
* JavaScript
* HTML
* CSS
* Algorithms
* Visulisation
